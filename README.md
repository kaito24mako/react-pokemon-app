# 🔎 Search 'em All!

A PokemonTCG search engine in which users can search for cards and add it to their collection for tracking purposes 

## ⚙️ Skills

**SASS**

- My first project using SCSS
- SASS variables, nesting, and inheritance
- Complex animation 

**React**

- Also my first project using React
- Components to segment aspects of the website
- Passing props
- useState and useEffect to allow for a dynamic re-rendering of website components
- useContext to allow for global prop use 
- React Router to enable re-rendering page links
- Use of various UI libraries, such as React-Spinners and Toastify
